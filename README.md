Project Description
This is a modern, fully responsive personal portfolio website built using HTML5, CSS3, and vanilla JavaScript. The portfolio showcases my skills, services, and projects through a clean, professional interface with smooth animations and interactive features.
Key Features:

Three-page structure (Home, Skills, Projects) with seamless navigation
Animated skill progress bars displaying technical proficiencies
Project filtering system by technology category
Light/Dark theme toggle for enhanced user experience
Comprehensive form validation for contact submissions
Session timer tracking user engagement
Mobile-responsive design with hamburger menu
Smooth scroll animations and hover effects

Technologies Used:

HTML5 for semantic structure
CSS3 with Flexbox/Grid for responsive layouts
Vanilla JavaScript for interactivity
Google Fonts for custom typography
Bootstrap Icons for visual elements

Purpose:
This portfolio serves as a comprehensive showcase of my web development capabilities, demonstrating proficiency in frontend technologies and modern design principles. It provides potential clients and employers with detailed information about my skills, completed projects, and professional background.
